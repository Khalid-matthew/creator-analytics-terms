# Terms of Service

This application provides analytics dashboards for content creators.

Creators voluntarily authorize the application to access their own
TikTok and Instagram account metrics through official platform APIs.

The application only accesses data explicitly permitted by the user
and does not collect data from non-authorized accounts.

All data is used solely for analytics and visualization purposes.
We do not sell, share, or misuse user data.

Users may revoke access at any time through the respective platform.

## Privacy

We store access tokens securely and only for the purpose of retrieving
authorized metrics. No personal data is shared with third parties.
